# Keymouse
Use your arrows to move your mouse without a physical one
# WARNING!
This python code requires keyboard and mouse libraries (pip install keyboard mouse)
## Notice
If the keys aren’t suppressed for system use (so arrows and other bindings won’t work normally), run the program as an administrator
## CONTROLS

Arrow Keys: Mouse Movement
Left Shift: Slow/Precision Movement
Right Shift: Left Click
Menu Key: Right Click
Page Up: Mouse Wheel Up
Page Down: Mouse Wheel Down
ESC: Safely exit
